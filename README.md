# massMatchR

**massMatchR** is an interactive R Shiny application for matching mass spectrometry (MS) peak lists against a user-defined reference database. It enables researchers to identify compounds across multiple samples simultaneously, visualize relative or absolute signal intensities and annotate bar charts with molecular structures.

massMatchR can be accessed freely at:

- http://dublin.embnet.sk:3838/massMatchR/

It can also be installed and run locally if modifications are required, for example, when the provided image list is not suitable for a specific use case.
